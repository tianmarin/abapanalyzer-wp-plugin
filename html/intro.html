Un Análisis de Performance y Configuración de un sistema SAP ABAP NetWeaver es un proceso que involucra la revisión detallada de información histórica de los diferentes componentes de un SAP AS ABAP. El comportamiento de diferentes variables como utilización de segmentos de memoria, buffers, workprocess, y parámetros debe ser evaluado por especialistas de cada rama involucrada.

2

La información básica de este tipo de análisis puede ser extraído directo de la aplicación SAP. Sin embargo para eso es necesario configurar correctamente la recolección de información.
Para esto debes seguir los siguientes pasos:

  • Habilitación y parametrización del Snapshot Monitoring
  • Habilitación y parametrización del Tune Summary
  • Habilitación y parametrización del Operating System Monitor
